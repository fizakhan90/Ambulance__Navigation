Welcome to Ambulance Navigation!

Ambulance Navigation System is a website created for emergency situations. The website focuses on two things.

1.On the Patients Side :

The patient will be able to call a nearby ambulance for aid and they will be able to track it on its way. The ambulance nearest to the patient will be dispatched to minimise the waiting time in emergency situations.

2.On the Ambulance Driver's Side:

The call will be requested with details of the destination displayed on the map the website along with the shortest path to the destination with the minimum traffic route and other features too

The project makes use of Mappls mapmyindia API for the purpose of all map related operations.
